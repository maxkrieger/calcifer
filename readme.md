# Calcifer

## Getting started

- Install npm packages

```
npm install
```

- Install `parcel`:

```
npm install -g parcel-bundler
```

- Run

```
npm start
```

- Navigate to [`http://localhost:1234/index.html`](http://localhost:1234/index.html)
